# ⚽ Elfmeter WM 2026 — Penalty Shootout

Ein 3D-Elfmeterschießen für den Browser – komplett in HTML, CSS und JavaScript.
Wähle dein Land und spiele dich im Turnier bis ins Finale.

## 🎮 So öffnest du das Spiel

**Variante 1 – Online (am einfachsten):**
Klicke einfach auf den Link und spiele sofort im Browser:

👉 **https://josyskd.github.io/soccer-web-spiel/**

Funktioniert auf Computer, Tablet und Handy – nichts installieren nötig.

**Variante 2 – Lokal auf deinem Gerät:**

1. Lade das Projekt herunter (grüner **`Code`**-Button oben → **Download ZIP**) und entpacke es.
2. Öffne den Ordner und mache einen Doppelklick auf die Datei **`index.html`**.
3. Das Spiel startet automatisch in deinem Browser.

## Steuerung

- **Land wählen** → tritt für deine Nationalmannschaft an
- **Schießen**: klicken/halten und in Richtung Tor zielen (am Desktop mit der Maus, am Handy per Swipe)
- **Halten**: du schießt und hältst abwechselnd – das Timing entscheidet
- Gewinne das Elfmeterschießen und ziehe eine Runde weiter, bis ins Finale

## Technik

- Vanilla JavaScript, 3D-Grafik mit [Three.js](https://threejs.org/)
- Länder-Trikots und -Flaggen prozedural in Landesfarben generiert
- Sound-Feedback bei Klicks und Aktionen
- Läuft auf Desktop und Handy (Touch/Swipe wird unterstützt)
